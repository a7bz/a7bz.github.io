# 本站 Markdown 使用示例

展示了本站支持的 Markdown 扩展语法和特性

## 一、基础语法

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

### 文本样式

```markdown
**粗体文本**
*斜体文本*
***粗斜体文本***
~~删除线~~
<u>下划线</u>
`行内代码`
```

**粗体文本**

*斜体文本*

***粗斜体文本***

~~删除线~~

<u>下划线</u>

`行内代码`

### 列表

#### 无序列表

```markdown
- 列表项 1
- 列表项 2
  - 子列表项 2.1
  - 子列表项 2.2
```

- 列表项 1
- 列表项 2
  - 子列表项 2.1
  - 子列表项 2.2

#### 有序列表

```markdown
1. 第一项
2. 第二项
   1. 子项 2.1
   2. 子项 2.2
```

1. 第一项
2. 第二项
   1. 子项 2.1
   2. 子项 2.2

### 引用

```markdown
> 这是一段引用文本
> 
> 引用可以换行
```

> 这是一段引用文本
> 
> 引用可以换行

### 代码块

#### 普通代码块

```
# 这是一段普通代码
print("Hello, World!")
```

#### 语法高亮代码块

```python
# 这是一段 Python 代码
print("Hello, World!")

def add(a, b):
    """加法函数"""
    return a + b
```

```javascript
// 这是一段 JavaScript 代码
console.log("Hello, World!");

const sum = (a, b) => a + b;
```

## 二、扩展语法

### 1. 自定义容器

#### 信息框 (Info)

```markdown
::: info 信息标题
这是一条信息提示
:::
```

::: info 信息标题
这是一条信息提示
:::

#### 提示框 (Tip)

```markdown
::: tip 提示标题
这是一条提示信息
:::
```

::: tip 提示标题
这是一条提示信息
:::

#### 警告框 (Warning)

```markdown
::: warning 警告标题
这是一条警告信息
:::
```

::: warning 警告标题
这是一条警告信息
:::

#### 危险框 (Danger)

```markdown
::: danger 危险标题
这是一条危险信息
:::
```

::: danger 危险标题
这是一条危险信息
:::

### 2. 标签页 (Tabs)

```markdown
::: tabs

== 标签页 1

== 标签页 2

内容 2

:::
```

::: tabs

== 标签页 1

内容 1

:::

```markdown

:::

未选中状态
:::
```

:::
